## DMIT 是什么

> **DMIT**（全称 DMIT Cloud Inc.）是一家成立于 2018 年、总部位于美国洛杉矶的云服务商，在香港、洛杉矶、东京、圣何塞等地部署了数据中心，核心卖点是直连中国大陆的优质线路——尤其是 CN2 GIA 和 CMI 双向回程。

香港节点是 DMIT 产品线里最受中文用户关注的部分，原因很简单：香港到大陆的物理距离加上 CN2 GIA 线路，能在不绕路的前提下给大陆用户提供接近本地的访问速度。

---

## DMIT 香港线路有什么不同

DMIT 香港目前主要提供两条线路方向：

**Premium（高端线路）**
- 三网回程 CN2 GIA（电信）+ CMI（移动）+ NT/PCCW（联通）
- 去程同样走优质路由，不绕美国
- 适合对延迟和稳定性要求高的业务：直播推流、游戏加速、企业内网穿透

**Economy（经济线路）**
- 回程走普通 BGP，带宽更大、价格更低
- 适合对延迟不敏感、以流量为主的场景：静态资源分发、备份节点

两条线路共用同一个香港数据中心（Equinix HK1），硬件层面没有差异，差的是出口路由策略。

## 我的实际使用体验

用 Premium 套餐跑了大约三个月，主要场景是部署一个面向大陆用户的 API 服务。

延迟方面，深圳到香港节点的 ping 值稳定在 3–8ms，上海在 25–35ms 区间，北京偶尔会到 45ms 但很少超过 50ms。晚高峰（20:00–23:00）延迟波动不超过 10ms，这是 CN2 GIA 线路的典型表现——电信骨干网在这个时段有专属通道，不会和普通 BGP 流量抢带宽。

丢包率在连续 72 小时的MTR 测试里基本维持在 0%，偶发的单跳丢包出现在 PCCW 的香港本地段，不影响端到端质量。

控制面板用的是自研系统，不是 SolusVM 或 Virtualizor，界面简洁但功能够用：一键重装、VNC 控制台、流量统计、IP 管理都在。工单响应速度中等，普通问题 2–4 小时内有人工回复，紧急问题标记 urgent 后通常 30 分钟内有响应。

一个小缺点：Premium 套餐的月流量上限相对保守，如果业务有突发流量需求，需要提前评估是否够用，或者选择更高档位。

## Premium 还是 Economy：怎么选

**选 Premium 的场景：**
- 面向大陆用户的 SaaS 产品、API 服务、直播推流
- 对延迟敏感的游戏服务器或实时通信
- 企业内网穿透、跨境办公 VPN 出口
- 月流量在 2TB 以内、愿意为稳定性付溢价

**选 Economy 的场景：**
- 静态资源 CDN 回源节点
- 备份服务器、日志归档
- 个人博客、低流量测试环境
- 预算有限、延迟不是核心指标

一个判断标准：如果你的业务有真实的大陆终端用户，Premium 的溢价通常值得——CN2 GIA 在晚高峰的表现和普通 BGP 差距肉眼可见，用户流失率和加载速度直接挂钩。

## FAQ

### DMIT 香港 VPS 支持退款吗？

新开通的 VPS 在 72 小时内可以申请无理由退款，退款金额退回原支付渠道。超过 72 小时后按剩余天数比例退款，具体以工单沟通为准。

### DMIT 香港的 IP 是否容易被封？

Premium 套餐使用的是 CN2 GIA 商业 IP 段，不是共享的数据中心 IP 池，被封概率低于普通 VPS 服务商。如果 IP 被封，可以提工单申请更换，通常收取小额费用。

### 香港节点支持 IPv6 吗？

Premium 套餐默认分配 1 个 IPv4，IPv6 需要在开通后通过控制面板申请，部分套餐免费提供 /64 段。

### 流量超出后会怎样？

不会直接断线，带宽会被限速到 1 Mbps 左右，直到下个计费周期重置。如果业务对流量需求不稳定，建议选高一档的套餐留出余量。

### DMIT 和搬瓦工香港节点怎么选？

两家都走 CN2 GIA，DMIT 的控制面板自研、IP 质量更稳定，搬瓦工的价格体系更透明、有年付大折扣。如果预算充足且对 IP 质量要求高，DMIT 更合适；如果追求性价比，搬瓦工的 CN2 GIA 套餐也是成熟选择。

### 支持 Windows 系统吗？

目前 DMIT 香港节点不提供 Windows 镜像，仅支持 Linux 发行版（Debian、Ubuntu、CentOS、AlmaLinux 等）。
